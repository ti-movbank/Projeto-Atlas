Como usar: As skills são módulos reutilizáveis que estendem as funcionalidades do Hermes Agent. Elas podem ser acionadas via hermes skill run <nome> ou agendadas em cron jobs para automação.

Exemplos populares: daily-cost-report (relatório de custos LLM), fiscal-news-fetcher (notícias fiscais), google-workspace (integração Gmail/Drive), notion (bases de conhecimento), gateway-log-auditor (auditoria de logs).

⚙️ Como Funciona

Processo de Atualização Automática

  1. O Hermes Agent executa tarefas agendadas (cron jobs) em horários pré-definidos
  2. Cada cron job coleta informações específicas do sistema ou executa tarefas solicitadas
  3. Os resultados são processados e formatados para exibição web
  4. As atualizações são commitadas e enviadas automaticamente para o repositório GitHub
  5. O GitHub Pages publica as alterações em tempo real, disponíveis para acesso público

Vantagem: Você acessa as informações atualizadas sem precisar de senhas, túneis ou acesso direto ao servidor.
